LPAF

Layered Presence Authentication Framework

Attendance is no longer claimed — it is cryptographically verified. A cybersecurity-driven attendance verification system preventing proxy submissions using dynamic QR tokens and network validation.

Why Current Attendance Systems Fail

Static QR Codes

Shared QR codes allow proxy attendance from anywhere.

No Physical Verification

Existing systems cannot verify that students are physically present.

Data Manipulation

Attendance records can easily be falsified.

The LPAF Security Stack

Dynamic QR Tokens

QR codes change every 90 seconds preventing reuse.

Network Validation

Submissions must originate from the same campus network.

Session Authentication

Each session generates unique cryptographic tokens.

Duplicate Prevention

Roll numbers cannot submit attendance twice.
